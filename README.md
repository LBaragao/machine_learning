# Previsão de Score de Crédito com Machine Learning

Projeto desenvolvido durante a **Jornada Python da Hashtag Treinamentos** com o objetivo de construir um modelo de Machine Learning capaz de prever o score de crédito de clientes.

## 📌 Descrição

Neste projeto foi desenvolvido um modelo de aprendizado de máquina que analisa características dos clientes e prevê seu score de crédito.

O objetivo é demonstrar como algoritmos de Machine Learning podem ser utilizados para apoiar decisões em cenários financeiros.

## ⚙️ Tecnologias utilizadas

- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## 🧠 Etapas do projeto

1. Importação da base de dados
2. Tratamento e preparação dos dados
3. Separação entre dados de treino e teste
4. Treinamento de modelos de Machine Learning
5. Avaliação da performance dos modelos

## 📊 Resultado

O modelo treinado foi capaz de prever o score de crédito dos clientes com boa taxa de acerto a partir das características presentes na base de dados.

## 📚 Aprendizados

- Fundamentos de Machine Learning
- Treinamento de modelos preditivos
- Avaliação de modelos
- Preparação de dados para algoritmos de ML
